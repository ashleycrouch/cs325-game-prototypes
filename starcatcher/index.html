  • Prototype Postmortem: I learned more about generating group objects and timers. I learned that Phaser doesn't really have a way to apply listeners for things that happen to an individual object from a group, so how you do that is that when they're spawned individually, you add the properties to the object on its own. It's a little silly to do it this way, but it was the easiest method I could find.
